Linked List

A dynamic data structure with efficient insertions and deletions.

Introduction

A Linked List is a linear data structure where elements are stored in nodes. Each node contains two parts: data and a reference (pointer) to the next node. Unlike arrays, linked lists do not store elements in contiguous memory locations.

Types of Linked List

Following are the various types of linked list.

  • Singly Linked List

    • Singly linked lists contain two "buckets" in one node; one bucket holds the data and the other bucket holds the address of the next node of the list. Traversals can be done in one direction only as there is only a single link between two nodes of the same list.

  • Doubly Linked List

    • Doubly Linked Lists contain three "buckets" in one node; one bucket holds the data and the other buckets hold the addresses of the previous and next nodes in the list. The list is traversed twice as the nodes in the list are connected to each other from both sides.

  • Circular Linked List

    • Circular linked lists can exist in both singly linked list and doubly linked list.Since the last node and the first node of the circular linked list are connected, the traversal in this linked list will go on forever until it is broken.

Traversal

Displaying the contents of a linked list is very simple. We keep moving the temp node to the next one and display its contents.

  1. Initialize a temporary node (temp) to point to the head of the linked list.
  2. While temp is not NULL, do the following:
    • Print the data of the current node (temp).
    • Move temp to the next node in the list (temp = temp->next).
struct node *temp = head; while (temp != NULL) { printf("%d ", temp->data); temp = temp->next; }

Insertion

adds a new element to the linked list

Insert at Beginning

  1. Allocate memory for new node
  2. Store data
  3. Change next of new node to point to head
  4. Change head to point to recently created node
struct node *newNode; newNode = malloc(sizeof(struct node)); newNode->data = 4; newNode->next = head; head = newNode;

Insert at End

  1. Allocate memory for new node
  2. Store data
  3. Traverse to last node
  4. Change next of last node to recently created node
struct node *newNode; newNode = malloc(sizeof(struct node)); newNode->data = 4; newNode->next = NULL; struct node *temp = head; while(temp->next != NULL){ temp = temp->next; } temp->next = newNode;

Insert at Given Position

  1. Allocate memory for new node
  2. Store data
  3. Traverse to position - 1
  4. Adjust pointers
temp = head for i = 1 to pos-1 temp = temp.next newNode.next = temp.next temp.next = newNode

Deletion

removes the existing elements

Delete from Beginning

  1. Store head node in a temporary variable
  2. Change head to point to next node
  3. Free memory of temporary variable
head = head.next

Delete from End

  1. Traverse to second last node
  2. Change next of second last node to NULL
  3. Free memory of last node
temp = head while temp.next.next != NULL temp = temp.next temp.next = NULL

Delete from Given Position

  1. Traverse to position - 1
  2. Change next of position - 1 node to point to position + 1 node
  3. Free memory of node at given position
temp = head for i = 1 to pos-1 temp = temp.next temp.next = temp.next.next

Reverse a Linked List

Reversing a linked list means reversing the direction of pointers so the list is traversed from last node to first.

  1. Initialize three pointers: prev as NULL, curr as head, and next as NULL
  2. Iterate through the linked list
  3. In each iteration, store the next node, reverse the current node's pointer, and move the pointers one position ahead
  4. Finally, update head to point to prev
prev = NULL curr = head while curr != NULL next = curr.next curr.next = prev prev = curr curr = next head = prev

Time Complexity Comparison

Operation Time Complexity
Insertion (Beginning) O(1)
Insertion (End) O(n)
Deletion O(n)
Traversal O(n)
Reverse O(n)
